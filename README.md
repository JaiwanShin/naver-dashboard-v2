# 📊 네이버 시장 트렌드 대시보드

네이버 검색 데이터를 활용한 **시장 분석 및 마케팅 인사이트 도구**입니다.

---

## 🚀 대시보드 접속

> 대시보드 URL은 보안상 별도로 공유됩니다. 담당자에게 문의하세요.

---

## 📋 탭별 기능 안내

| 탭 | 기능 | 주요 활용 |
|---|------|----------|
| 🔍 **키워드 트렌드** | 검색량 추이 분석 | 관심 키워드의 월별/주별 트렌드 확인, **3개월 예측** |
| 🛒 **쇼핑 트렌드** | 카테고리 클릭 트렌드 | 화장품/가전 등 카테고리별 소비자 관심도 변화 |
| 📦 **상품 검색** | 경쟁사 상품 조회 | 가격 분포, 브랜드/판매처 점유율 분석 |
| ⚔️ **경쟁 비교** | 다중 브랜드 비교 | 브랜드별 검색량/가격/고객층 비교 |
| 📈 **성별/연령 분석** | 타겟 고객 분석 | 성별·연령대별 검색 비율 확인 |
| 🔑 **키워드 리서치** | 마케팅 키워드 발굴 | 연관 키워드 100개+ 발굴 |
| 🚀 **시장 진입 분석** | 신규 진출 검토 | 시장 규모, 경쟁 강도, 진입 장벽 지수 |
| 📊 **실제 검색량** | 월간 검색량 조회 | PC/모바일 실제 검색 횟수 확인 |

---

## 💡 주요 기능 사용법

### 1. 키워드 트렌드 분석
1. **키워드 입력**: 분석하고 싶은 키워드를 쉼표로 구분하여 입력 (최대 5개)
2. **조회 기간 설정**: 좌측 사이드바에서 시작일/종료일 선택
3. **🔎 트렌드 분석** 버튼 클릭
4. 결과: 트렌드 그래프, 요약 통계, **3개월 후 예측**

### 2. 경쟁 브랜드 비교
1. "경쟁 비교" 탭 선택
2. 비교할 브랜드명 입력 (예: `캄프, 메디힐, 라운드랩`)
3. 분석 모드 선택:
   - **검색 트렌드**: 브랜드별 검색량 추이
   - **고객층 분석**: 성별/연령대 비교
   - **상품 가격**: 가격대 비교 (중앙값 기준)

### 3. 마케팅 키워드 발굴
1. "키워드 리서치" 탭 → **연관 키워드 + 검색량** 모드
2. 시드 키워드 입력 (예: `토너패드, 스킨케어 패드`)
3. 결과: **100개 이상**의 연관 키워드 + 월간 검색량 + 경쟁도

### 4. 데이터 다운로드
- 각 분석 결과 하단의 **📊 Excel 리포트 다운로드** 버튼 클릭
- 피벗 테이블, 요약 통계 등이 포함된 Excel 파일 저장

## 📊 데이터 해석 가이드

| 지표 | 설명 |
|------|------|
| **검색지수 (0~100)** | 조회 기간 내 가장 높은 검색량을 100으로 설정한 상대값 |
| **중앙값** | 전체 데이터의 중간 지점 (평균보다 이상치에 강함) |
| **경쟁도** | 높음/중간/낮음 - 해당 키워드의 광고 경쟁 수준 |
| **이상치 제외** | 극단적인 가격(미끼 상품 등)을 자동 제거한 통계 |

---

## ⚠️ 주의사항

- **상대값 vs 절대값**: "키워드 트렌드" 탭은 상대값(0~100)이며, 실제 검색 횟수는 "실제 검색량" 탭에서 확인
- **데이터 갱신**: 네이버 API 데이터는 실시간이 아니며, 보통 1~2일 지연됨
- **API 한도**: 일 1,000회 호출 제한이 있으므로 과도한 조회 시 일시 중단될 수 있음
